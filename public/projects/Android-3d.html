Android 3D Modeling

3D & Animation | 2020

Android is a mobile operation system developed by Google and their mascot is a green robot. This robot has an unofficial nickname of Bugdroid. Bugdroid has long been a symbol of the fun and curiosity at Android. As I researched more about the Android logo, I found out that Google actually re-branded their logo last year. They changed the font, colour and symbol. The design has become more simple and minimal with a trendy colour and a modern font. In 2019 version, The robot’s body is missing and only its head is left. I decided to make both the old Bugdroid and a new logo in 3ds Max. So the idea that came to my mind was the scene where the old Bugdroid giving a presentation of a new logo for Android.

3D RENDERING

I used the official colour value of Android. As for the texture, I decided to go with a satin finish. But I made the new robot little bit more glossy than the old one because glossiness made the new robot more trendy and fits well to the current colour.

PROCESS - REFERENCE IMAGE

In order to make the Bugdroid model in 3ds Max, I used my robot figure as a reference. Because it is already a 3D figure, it helps me to create a 3D model easier. As for the new logo, I used below picture as a reference. I imported the picture and used it as a backdrop so that I can scale the logo properly. As for the type style, I downloaded Galano Grotesque DEMO Bold from the Internet.

PROCESS - COLOUR

I searched for the RGB colour of both Android’s old logo and the new logo. I was simply able to find them in Google. However, I realized that I need to convert the general RGB colour value into a 3ds Max version because the value that I found was 0-255 scale value while 3ds Max only has 1-0 scale. I converted the colour data using the convert website called Easy RGB calculator.
